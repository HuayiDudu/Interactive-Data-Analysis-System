# 可视化自定义参数 — Web 界面接入说明

**编写日期**：2026-06-04
**状态**：待 Web 界面模块开发人员实现

---

## 背景

后端 `VisualizeService.generate_plot()` 已新增 `**options` 自定义参数支持，所有硬编码样式值改为可覆盖的默认值。**当前前端未传入任何自定义参数，全部走后端默认值，功能不受影响。**

此文档说明 Web 界面需要做的改动，以便用户能在 UI 上调节图表样式。

---

## 后端支持的参数清单

| 参数 | 类型 | 默认值 | 适用图表 | 说明 |
|------|------|--------|---------|------|
| `title` | string | 自动生成 | 全部 | 图表标题，空字符串或 null 则自动生成 |
| `color_scheme` | list | D3 配色 | 全部 | Plotly 颜色序列，如 `px.colors.qualitative.Set1` |
| `opacity` | float | `0.8` | scatter | 散点透明度，范围 0~1 |
| `marker_size` | int | `12`(scatter) / `10`(line) | scatter, line | 标记点大小 |
| `aggregation` | string | `"mean"` | bar | 聚合方式：`mean` / `sum` / `count` / `median` |
| `pie_hole` | float | `0.3` | pie | 饼图空心比例，0 为实心饼图 |
| `pie_max_categories` | int | `10` | pie | 饼图最多显示的分类数，超出归为"其他" |

---

## 需要 Web 界面做的事

### 1. 在可视化面板添加自定义控件

在 `templates/index.html` 的 `#visualize-view` 区域，现有 `plot-x` / `plot-y` / `plot-type` 三个控件之后，增加自定义参数控件。建议：

- **标题输入框**：`<input id="plot-title" type="text" placeholder="留空自动生成">`
- **散点图透明度**：`<input id="plot-opacity" type="range" min="0" max="1" step="0.1" value="0.8">`
- **标记点大小**：`<input id="plot-marker-size" type="range" min="4" max="30" value="12">`
- **柱状图聚合方式**：`<select id="plot-aggregation"><option value="mean">均值</option><option value="sum">总和</option>...</select>`
- **饼图空心比例**：`<input id="plot-hole" type="range" min="0" max="0.8" step="0.1" value="0.3">`

> **设计建议**：这些控件可以放在一个折叠面板（`<details>` / Bootstrap collapse）或"高级设置"区域中，避免默认界面过于拥挤。

### 2. 修改 `static/js/plot.js` 的 `handlePlot()`

在构造请求 body 时，读取自定义控件的值并加入 JSON：

```javascript
// 现有必填字段
const body = {
    dataset_id: datasetId,
    x: xCol,
    y: yCol,
    type: plotType
};

// 读取自定义参数（只传有值的，让后端走默认）
const title = document.getElementById('plot-title')?.value?.trim();
if (title) body.title = title;

const opacity = document.getElementById('plot-opacity')?.value;
if (opacity && parseFloat(opacity) !== 0.8) body.opacity = parseFloat(opacity);

const aggregation = document.getElementById('plot-aggregation')?.value;
if (aggregation && aggregation !== 'mean') body.aggregation = aggregation;

// ... 其他参数同理
```

### 3. 联动逻辑

- **图表类型切换时**：显示/隐藏对应的自定义控件。例如选 pie 时才显示 `pie_hole` 滑块，选 bar 时才显示聚合方式下拉框，选 scatter 时才显示透明度滑块。
- **切换图表类型不应重置**已填写的自定义参数值（与 XY 下拉框同样的处理原则）。

### 4. 可选：`populatePlotColumns()` 相关

无需改动。自定义参数与 X/Y 列选择是正交的。

## 验收标准

- [ ] 不填任何自定义参数时，生成的图表与改动前完全一致
- [ ] 修改标题输入框后，图表标题变为用户输入的内容
- [ ] 调节透明度/标记大小后，散点图样式相应变化
- [ ] 切换聚合方式后，柱状图数值变化（mean vs sum 结果不同）
- [ ] 调节饼图空心比例后，饼图内环大小变化
- [ ] 切换图表类型时，只有相关的自定义控件可见
- [ ] 后端对非法参数值返回 400 错误，前端应显示错误提示而非静默失败
